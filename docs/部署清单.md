## 二、部署步骤

### 步骤 1: 上传文件

```bash
# 在本地执行 (或用 U盘 / scp)
scp -r voice_assistant/ root@10.220.223.222:/home/sunrise/ucar_01/
scp -r src/ucar_vision/ root@10.220.223.222:/home/sunrise/ucar_01/src/
scp start_all_slam.sh ucar_nodes.service root@10.220.223.222:/home/sunrise/ucar_01/
```

### 步骤 2: 编译

```bash
ssh root@10.220.223.222
cd /home/sunrise/ucar_01
source /opt/tros/humble/setup.bash

# 编译所有包
colcon build --packages-select ucar_vision voice_assistant jgb520_driver rplidar_ros
```

### 步骤 3: 安装系统服务

```bash
cp /home/sunrise/ucar_01/ucar_nodes.service /etc/systemd/system/
systemctl daemon-reload
systemctl enable ucar_nodes.service
```

### 步骤 4: 重启测试

```bash
reboot
# 等待 60 秒后检查
tail -f /tmp/ucar_logs/main.log
```
